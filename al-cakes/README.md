# A&L Cakes 🎂

Site fictício de confeitaria artesanal — projeto acadêmico em **HTML, CSS e JavaScript**.

## 📁 Estrutura

```
al-cakes/
├── index.html          # Página inicial
├── css/
│   └── style.css       # Folha de estilos principal
├── js/
│   ├── script.js       # Interações (menu, validações, animações)
│   └── produtos.js     # Catálogo e renderização dos produtos
├── images/             # Imagens dos produtos e do site
└── pages/
    ├── cardapio.html   # Lista de produtos
    ├── produto.html    # Detalhe de um produto (?id=...)
    ├── blog.html       # Galeria estilo Instagram + posts
    ├── sobre.html      # História, missão e valores
    ├── login.html      # Tela de login
    ├── cadastro.html   # Cadastro de cliente
    └── admin.html      # Cadastro de produto (simulação)
```

## ▶️ Como executar

Basta abrir o arquivo **`index.html`**.

## 🎨 Paleta de cores

- Rosa suave: `#f8c8dc`
- Rosa claro: `#fce4ec`
- Rosa escuro (acento): `#d97aa3`
- Bege fundo: `#fdf8f5`
- Branco: `#ffffff`

## ✨ Recursos

- Layout responsivo (desktop + mobile com menu hambúrguer)
- Animações suaves ao rolar (IntersectionObserver)
- Validação de formulários (login, cadastro e admin)
- Catálogo dinâmico renderizado por JavaScript
- Compra integrada via link `wa.me` (WhatsApp)
